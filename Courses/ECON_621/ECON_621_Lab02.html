Each lab assignment needs to be submitted as the html output of an R Markdown file. At the top of each submission should be the title (eg, ECON 621 - Lab #1) and the student's name. File names should use the following structure: StudentLastName_ECON621_Lab_LabNumber. For example, Gorski_ECON621_Lab_01.

Assignments are due before the beginning of the following class.

[Before starting an R Markdown file] Create a folder on your computer for this course (eg, "/Users/studentname/ECON_621"). Set your working directory in R Studio to that folder.

1. Demonstrate that you have set your working directory to your course folder with getwd().

2. Go to this website and download the .csv file of historical agricultural and economic data. Save it in your course folder. Import the data into your R environment and call the dataframe ag_data. Print the first row of ag_data.

3. Connect to the course database and import the airports dataset with this SQL query: 

SELECT * FROM econ_621.airports

Call the dataframe airports. Print the first row of airports.

4.

a) Create a vector of the first ten values in the city column of airports. Call the vector cities and print it.

b) Subset cities to return every value except "Mount Hagen" and "Godthaab." Use positive integers to do so.

c) Subset cities to return every value except "Mount Hagen" and "Godthaab." Use negative integers to do so. 

d) Subset cities to return every value except "Mount Hagen" and "Godthaab." Use a logical vector to do so.

e) Subset cities to return every value except "Mount Hagen" and "Godthaab." Use a conditional statement to do so.

5. Print the twenty-forth row, second column of airports.

6. Print the latitude and longitude of Trombetas Airport in Brazil.

7. Print the IATA call signs for all the airports in Zambia.

8. Use conditional statements to print a subset of ag_data for Food Aid (Commodity value) to Nepal in 1990.

9. Use conditional statements to print all the years from ag_data where the population of Laos was less than 4 million.

10.

a) Use conditional statements to create a subset of ag_data with all the rows for El Salvador, Guatemala, Honduras, and Nicaragua. Call the subset cent_amer and print the first row.

b) Print a two-column table with columns Country and Amount, that ranks the countries by their 1998 Root Crop Yield, least to most.

c) Create a subset of cent_amer with all of the economic data from 2010 on. Call the subset cent_amer_econ. Convert all the percent change values to their actual value (ie, divide by 100). Print the modified rows with their updated values.
