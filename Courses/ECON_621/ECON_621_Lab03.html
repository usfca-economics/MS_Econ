Each lab assignment needs to be submitted as the html output of an R Markdown file. At the top of each submission should be the title (eg, ECON 621 - Lab #1) and the student's name. File names should use the following structure: StudentLastName_ECON621_Lab_LabNumber. For example, Gorski_ECON621_Lab_01.

Assignments are due before the beginning of the following class.

Start by downloading the airports dataset from the course database with this query:

SELECT * FROM econ_621.airports

1. Print the dimensions of airports.

2. What is the data class of of each of the columns ? Are there any that should or could be considered category variables? Why?

3. Write a script that:

a. Defines a vector of required columns as "name", "latitude", "longitude", and "avg_flights_per_day" 

b. Returns a T/F if all the columns are in airports

c. Returns the name(s) of the missing column(s)

4. Change the name of the timezone column to hrs_from_UTC.

5. Which columns contain one or more NA values?

6. Are there NA values in the city column? Are there no missing values there? Set all missing values to NA.

7. Use the summary function to describe the altitude of airports in this dataset.

8. Which airport in China is at the highest altitude?

9. Find the average hrs_from_UTC: can you infer anything from this value about the general location of airports in this dataset?

10. Are there any airports that are missing both a IATA or ICAO airport code? Use the table function and T/F conditional statements to answer.

11. Which is the country with the most airports? What percentage of all airports in airports are in that country?

12. Are there cities with more than one airport?

a. Before investigating, what do you expect to see as an answer?

b. Use the duplicated function to see whether there are any cities with more than one airport (hint: you'll need both the city and country columns).

c. What is the name (city and country) of one of the cities with more than one airport?

13. Merging airports and GDP data

a. Import the most recent GDP data from the natl_gdp dataset in the course database, using this query:

SELECT country, "2017" as "gdp_2017"
FROM econ_621.natl_gdp
WHERE "2017" IS NOT NULL

b. Create a new dataset by merging airports with your GDP data. Ensure that your dataset only contains rows with a value for GDP and an airport ID. Print the dimensions of this new dataset.

c. Create another new dataset, again by merging airports with your GDP data. This time, keep all data from either dataset.
